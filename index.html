üîß Basic Setup & Configuration

Config
git config --global user.name "Your Name"
Set your global username for all repositories
git config --global user.name "John Doe"
Config
git config --global user.email "email@example.com"
Set your global email address for commits
git config --global user.email "john@example.com"
Config
git config --global init.defaultBranch main
Set the default branch name to 'main'
Config
git config --list
Show all Git configuration settings
Config
git config --global core.editor "code --wait"
Set VS Code as the default Git editor
Help
git help <command>
Get help for a specific Git command
git help commit

üìÅ Creating & Managing Repositories

Init
git init
Initialize a new Git repository in current directory
Clone
git clone <repository-url>
Clone a remote repository to your local machine
git clone https://github.com/user/repo.git
Clone
git clone <url> <folder-name>
Clone repository into a specific folder
git clone https://github.com/user/repo.git my-project
Init
git init <project-name>
Initialize repository in a new directory
git init new-project

üìù Making Changes & Commits

Status
git status
Show the working directory status
Add
git add <file>
Add specific file to staging area
git add index.html
Add
git add .
Add all changes to staging area
Add
git add *.js
Add all JavaScript files to staging
Commit
git commit -m "commit message"
Commit staged changes with a message
git commit -m "Add user authentication"
Commit
git commit -am "message"
Add and commit in one command (only for modified files)
Commit
git commit --amend
Modify the most recent commit

üåø Branching & Merging

Branch
git branch
List all local branches
Branch
git branch <branch-name>
Create a new branch
git branch feature-auth
Checkout
git checkout <branch-name>
Switch to another branch
git checkout main
Checkout
git checkout -b <branch-name>
Create and switch to new branch
git checkout -b hotfix-bug
Merge
git merge <branch-name>
Merge specified branch into current branch
git merge feature-auth
Branch
git branch -d <branch-name>
Delete a branch (safe)
Branch
git branch -D <branch-name>
Force delete a branch

üîç Inspecting History

Log
git log
Show commit history
Log
git log --oneline
Compact commit history view
Log
git log --graph --oneline
Visual history with branch graph
Diff
git diff
Show unstaged changes
Diff
git diff --staged
Show staged changes
Show
git show <commit-hash>
Show specific commit details
git show abc123

‚Ü©Ô∏è Undoing Changes

Restore
git restore <file>
Discard changes in working directory
git restore script.js
Restore
git restore --staged <file>
Unstage a file
Reset
git reset <commit>
Reset to specific commit (keep changes)
Reset
git reset --hard <commit>
Reset and discard all changes
Revert
git revert <commit>
Create new commit that undoes changes
Clean
git clean -fd
Remove untracked files and directories

üåê Remote Repositories

Remote
git remote -v
List remote repositories
Remote
git remote add <name> <url>
Add a new remote repository
git remote add origin https://github.com/user/repo.git
Push
git push <remote> <branch>
Push commits to remote repository
git push origin main
Pull
git pull <remote> <branch>
Fetch and merge from remote
git pull origin main
Fetch
git fetch <remote>
Download objects from remote
Clone
git clone --branch <branch> <url>
Clone specific branch
git clone --branch develop https://github.com/user/repo.git

üë• Collaboration

Fetch

Pull
git pull --rebase
Pull with rebase instead of merge
Push

Push and set upstream branch
Stash
git stash
Temporarily save changes
Stash
git stash pop
Restore stashed changes
Branch
git branch -r
List remote branches

üí° Pro Tips

Use descriptive commit messages that explain WHY the change was made
Commit often - small, focused commits are easier to understand and revert
Use branches for new features and bug fixes to keep main stable
Always pull before you push to avoid merge conflicts
Use git status frequently to know where you are
Write good commit messages: present tense, capitalized, no period
